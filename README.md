# Notebolt User Guide

## Introduction
Welcome to Notebolt, your AI-powered lecture notes generator. This guide will help you get started with using Notebolt to transcribe and summarize your lecture audio files.

## Features
- **Transcription:** Convert lecture audio into text.
- **Summarization:** Highlight key points and generate concise summaries.
- **Q&A:** Ask questions about the generated notes through a chat interface.

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/davidkjeremiah/Notebolt.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Notebolt
    ```
3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage
1. Run the Streamlit app:
    ```bash
    streamlit run app.py
    ```
2. Upload your lecture audio file.
3. Adjust model parameters as needed.
4. Wait for the AI to process the lecture.
5. View the transcribed text and summarized notes.
6. Use the chat interface to ask questions about the notes.
7. Download the notes in text or PDF format.
